# QS-term
QS-terminal - GNU terminal for Unix-like system

![QS](https://github.com/user-attachments/assets/c488ea57-2ab1-4f71-aac7-7e9610c1c384)


# Install
`sudo pacman -S gtk3 vte3`

`make`

`sudo make install`

# Control

Ctrl + T - Change theme

Ctrl + R - reverse i search

Shift + Insert - pasta

Shift + Home - copy text
